# Knowledge Base Schema — Microbiome & Immunity

  

## Domain

Microbial diversity and its impact on immune system —

gut microbiome composition, immune regulation mechanisms,

clinical evidence, disease associations, interventions.

  

## Directory conventions

- wiki/concepts/    → 核心概念（每个概念一页）

- wiki/papers/      → 论文页（每篇论文一页）

- wiki/species/     → 关键菌种页（每个菌种一页）

- wiki/mechanisms/  → 作用机制页

- wiki/synthesis/   → 综述、疾病关联总结、时间线

## Key concepts to track

微生物组成: 多样性指数、α多样性、β多样性、菌群失调

免疫调节: Treg、Th17、IgA、肠道屏障、toll样受体

关键代谢产物: 短链脂肪酸、胆汁酸、吲哚

疾病关联: IBD、过敏、自身免疫、肿瘤免疫

干预手段: 益生菌、益生元、粪菌移植、饮食

  

## Ingest workflow

When asked to ingest new content:

1. Identify type: paper / concept / species / mechanism

2. Create corresponding wiki page following format above

3. Create any new concept/species/mechanism pages if needed

4. Add wikilinks between all related pages

5. Update index.md

6. Log in log.md
